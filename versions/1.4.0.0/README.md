## 功能概述

优化开机自启体验，程序启动后自动最小化到系统托盘，不占用任务栏空间。用户可通过托盘图标快速恢复窗口或退出程序。

### 计划功能

1. **系统托盘图标**：在系统托盘区域显示程序图标
2. **最小化到托盘**：窗口最小化时隐藏到托盘，不显示在任务栏
3. **托盘右键菜单**：右键托盘图标显示"显示窗口"和"退出"选项
4. **双击恢复**：双击托盘图标恢复窗口显示
5. **自启最小化**：开机自启时自动最小化到托盘
