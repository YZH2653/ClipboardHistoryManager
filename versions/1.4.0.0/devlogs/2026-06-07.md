### 2. 实现系统托盘功能

**时间**：框架创建后

**完成内容**：
- 添加 `<shellapi.h>` 头文件和 `WM_TRAYICON` 自定义消息
- 定义 `NOTIFYICONDATA` 全局变量 `G_Nid`
- 实现 `AddTrayIcon()` 函数：创建托盘图标
- 实现 `RemoveTrayIcon()` 函数：删除托盘图标
- 实现 `MinimizeToTray()` 函数：隐藏窗口到托盘
- 实现 `RestoreFromTray()` 函数：从托盘恢复窗口
- 在 `WM_SIZE` 中检测 `SIZE_MINIMIZED`，最小化时隐藏到托盘
- 在 `WindowProc` 中处理 `WM_TRAYICON` 消息：
  - 双击托盘图标恢复窗口
  - 右键托盘图标显示菜单（显示窗口/退出）
- 在 `WM_DESTROY` 中清理托盘图标
- 在 `main()` 中添加托盘图标，自启时最小化到托盘
- 更新版本号到 1.4.0.0

**技术细节**：
- 使用 `Shell_NotifyIcon` API 管理托盘图标
- 使用 `TrackPopupMenu` 显示右键菜单
- 使用 `SetForegroundWindow` 确保菜单正确显示
- 托盘图标优先从 exe 资源加载，失败时使用默认图标
