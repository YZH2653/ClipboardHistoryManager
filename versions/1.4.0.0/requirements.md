# 版本 1.4.0.0 需求文档

## 功能需求

### 1. 系统托盘图标

**目标**：在 Windows 系统托盘区域显示程序图标

**技术方案**：
- 使用 `Shell_NotifyIcon` API 创建托盘图标
- 使用 `NOTIFYICONDATA` 结构体配置图标
- 图标使用程序默认图标（从 exe 资源中加载）

**功能要求**：
- 程序启动时在托盘区域显示图标
- 图标提示文本显示"历史剪贴板管理器"
- 程序退出时删除托盘图标

### 2. 最小化到托盘

**目标**：窗口最小化时隐藏到系统托盘，不显示在任务栏

**技术方案**：
- 在 `WM_SIZE` 消息中检测 `SIZE_MINIMIZED`
- 最小化时调用 `ShowWindow(hWnd, SW_HIDE)` 隐藏窗口
- 从任务栏移除窗口（不使用 `WS_EX_TOOLWINDOW`，而是直接隐藏）

**功能要求**：
- 点击最小化按钮时，窗口隐藏到托盘
- 任务栏不再显示程序图标
- 托盘图标仍然可见

### 3. 托盘右键菜单

**目标**：右键点击托盘图标显示操作菜单

**技术方案**：
- 在 `WM_TRAYICON` 消息中处理 `WM_RBUTTONDOWN`
- 使用 `TrackPopupMenu` 显示弹出菜单
- 菜单项："显示窗口"、"退出"

**功能要求**：
- 右键托盘图标弹出菜单
- "显示窗口"：恢复窗口到前台显示
- "退出"：完全退出程序
- 点击菜单外部区域自动关闭菜单

### 4. 双击恢复窗口

**目标**：双击托盘图标恢复窗口显示

**技术方案**：
- 在 `WM_TRAYICON` 消息中处理 `WM_LBUTTONDBLCLK`
- 调用 `ShowWindow(hWnd, SW_SHOW)` 恢复窗口
- 调用 `SetForegroundWindow(hWnd)` 将窗口置前

**功能要求**：
- 双击托盘图标恢复窗口
- 窗口恢复后显示在前台
- 恢复后任务栏重新显示程序图标

### 5. 自启时最小化到托盘

**目标**：开机自启时程序自动最小化到托盘

**技术方案**：
- 在 `main()` 中读取自启设置
- 如果 `G_AutoStart` 为 `true`，启动时直接最小化到托盘
- 不显示主窗口，只显示托盘图标

**功能要求**：
- 开机自启时，程序启动后不显示主窗口
- 仅在托盘区域显示图标
- 用户可通过托盘图标恢复窗口

## 技术要求

1. 使用 Windows API `Shell_NotifyIcon` 管理托盘图标
2. 自定义窗口消息 `WM_TRAYICON` 处理托盘事件
3. 在 `WM_DESTROY` 中清理托盘图标
4. 编译时需要链接 `shell32` 库（托盘图标操作）
