### 实际更新内容

1. **SQLite 存储**：使用 SQLite 数据库替代 JSON 文件，解决数据丢失问题
2. **截图去重**：使用图片哈希比较，10秒内相同图片只保存一次
3. **图片保存修复**：修复 `SetRootDir` 未调用导致的图片保存失败
4. **移除调试输出**：删除控制台调试日志，添加 `-mwindows` 编译参数
