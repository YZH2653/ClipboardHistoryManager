### 遇到的问题

- **问题1**：历史记录丢失
  - **原因**：使用 JSON 文件存储，路径编码和文件操作有问题
  - **解决**：改用 SQLite 数据库存储

- **问题2**：截图重复保存（5次）
  - **原因**：系统截图工具多次更新剪贴板，每次序列号不同
  - **解决**：使用图片哈希比较，10秒内相同图片只保存一次

- **问题3**：图片保存失败（状态=7）
  - **原因**：`G_ClipManager.SetRootDir` 未调用，图片路径为空
  - **解决**：在 Main.cpp 中添加 `G_ClipManager.SetRootDir(exeDir)`

- **问题4**：程序运行弹出 cmd 窗口
  - **原因**：编译时缺少 `-mwindows` 参数
  - **解决**：在 tasks.json 中添加 `-mwindows` 参数
