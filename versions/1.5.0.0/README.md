## 功能概述

新增批量删除功能，用户可以全选或多选历史记录，然后一次性删除，提高管理效率。

### 计划功能

1. **全选功能**：添加全选复选框，一键选中所有记录
2. **多选功能**：每条记录前添加复选框，支持单选或多选
3. **批量删除**：选中多条记录后，点击删除按钮一次性删除
4. **选择计数**：显示已选中记录的数量
5. **确认对话框**：批量删除前显示确认提示，防止误删
