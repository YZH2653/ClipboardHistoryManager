## 遇到的问题

1. 编译 sqlite3.c 时需要单独作为 C 文件编译，不能直接与 C++ 源文件一起编译
   - 解决方案：先用 gcc 编译为 .o 文件，再与 C++ 源文件链接

## 技术笔记

- Windows 开机自启通过注册表 `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run` 实现
- 使用 `RegSetValueExW` 写入、`RegQueryValueExW` 读取、`RegDeleteValueW` 删除
- 编译时需要链接 `-ladvapi32`
- sqlite3.c 需要单独编译：`gcc -c -o sqlite3.o sqlite3.c`
