## 功能概述

新增开机自启动功能，用户可在设置中自由开关。

### 计划功能

1. **开机自启**：程序可设置为开机自动启动
2. **设置开关**：在设置页面添加开关按钮，可随时开启/关闭
3. **注册表管理**：通过 Windows 注册表实现自启动
