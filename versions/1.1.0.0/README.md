## 功能概述

新增设置页面，提供用户自定义配置能力。

### 新增功能

1. **设置按钮**：主界面新增设置按钮，点击切换到设置页面
2. **保存时间设置**：选项卡形式，支持 3天/5天/7天/30天/永久
3. **版本号页面**：显示版本号、更新日期、更新内容、作者信息
