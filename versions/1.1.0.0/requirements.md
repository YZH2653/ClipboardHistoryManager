# 版本 1.1.0.0 需求文档

## 功能需求

### 1. 设置按钮

**位置**：主界面右上角或底部工具栏
**交互**：点击后切换到设置页面
**图标**：齿轮图标 ⚙️

### 2. 设置页面

#### 2.1 保存时间设置

**展现形式**：选项卡/单选按钮
**选项**：
- 3天
- 5天
- 7天
- 30天
- 永久

**行为**：
- 选择后立即生效，保存到配置文件
- 下次启动时自动加载上次设置
- 影响后续过期清理的时间判断

#### 2.2 版本号页面

**入口**：设置页面中的"关于"或"版本信息"
**展现形式**：独立页面

**内容**：
```
版本号：1.1.0.0
更新日期：2026-06-01
更新内容：
  - 新增设置页面
  - 支持保存时间配置
  - 显示版本信息
作者：YZH2653
```

## 技术要求

1. 配置文件使用 JSON 格式存储
2. 设置页面需要支持返回主界面
3. 版本号需要与代码中的定义保持一致
4. 设置变更需要实时生效，无需重启程序

## 界面设计要求

1. 设置页面风格与主界面保持一致
2. 选项卡选中状态要有视觉反馈
3. 版本号页面排版清晰，信息层次分明
