# 开发日志

本文件夹记录项目的开发进度和待办事项。

## 日志格式

每个开发日志文件命名为：`YYYY-MM-DD-开发内容.md`

例如：`2024-01-15-基础框架搭建.md`

## 日志内容结构

```markdown
# 开发日志 - [日期]

## 今日完成
- [ ] 任务1
- [ ] 任务2

## 待办事项
- [ ] 任务3
- [ ] 任务4

## 遇到的问题
- 问题描述及解决方案

## 明日计划
- 计划任务列表
```
