# 历史剪贴板管理器

一个轻量级的Windows历史剪贴板管理软件，自动记录您复制的所有内容，方便随时查看和再次使用。

## ✨ 功能特性

- **自动记录**：复制的文字和图片会自动保存到本地
- **历史查看**：按时间倒序显示所有记录，最新复制的内容排在最前面
- **搜索功能**：支持文字搜索和时间搜索（日期、时间、年月）
- **置顶功能**：重要内容可以置顶，固定在列表顶部
- **删除功能**：不需要的记录可以手动删除
- **快速复制**：点击卡片即可复制内容到剪贴板
- **过期清理**：自动清理超过保留期限的记录
- **便携使用**：绿色软件，无需安装，可直接放在U盘使用

## 🚀 快速开始

### 编译运行

```bash
# 编译程序
g++ -std=c++17 -Wall -Wextra Main.cpp ClipboardManager.cpp Storage.cpp --output ClipboardHistory.exe -lgdiplus -lgdi32 -luser32

# 运行程序
./ClipboardHistory.exe
```

### 使用方法

1. 启动程序后，它会自动监听剪贴板变化
2. 复制任何文字或图片，程序会自动记录
3. 点击托盘图标打开主界面
4. 在界面中查看、搜索、管理历史记录

## 🔍 搜索格式

| 格式 | 示例 | 说明 |
|------|------|------|
| 完整日期 | `2026-05-31` | 搜索指定日期的记录 |
| 短日期 | `05-31` | 搜索月-日的记录 |
| 年月 | `2026-05` | 搜索指定年月的记录 |
| 时间 | `17:10` | 搜索指定时间的记录 |
| 文字 | `剪贴板` | 搜索包含关键词的记录 |

## 📁 项目结构

```
ClipboardHistory/
├── Main.cpp                 # 主程序（UI和逻辑）
├── ClipboardManager.h/cpp   # 剪贴板管理
├── Storage.h/cpp            # 存储管理
├── json.hpp                 # JSON库
├── clips/                   # 存储目录（运行时创建）
│   ├── history.json         # 历史记录索引
│   └── images/              # 图片存储
├── docs/                    # 项目文档
└── devlogs/                 # 开发日志
```

## 🛠️ 技术栈

- **语言**：C++17
- **编译器**：MinGW-w64 (GCC)
- **UI框架**：Win32原生API
- **JSON库**：nlohmann/json
- **图片处理**：GDI+

## 📋 系统要求

- Windows 7/8/10/11
- 无需额外安装任何组件

## 📦 便携版使用

1. 将 `portable/历史剪贴板管理器` 文件夹复制到U盘
2. 在其他电脑上，双击 `启动程序.bat` 或 `ClipboardHistory.exe`
3. 程序会自动创建 `clips` 文件夹存储数据
4. 数据跟随程序存储，换电脑不丢失

## ⚙️ 配置说明

- **保留天数**：默认保留最近3天的记录
- **最大记录数**：默认最多保存1000条记录
- **文字限制**：单条文字最多保存10000字符
